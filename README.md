
# Snake Water Gun Game 🐍💧🔫

A simple command-line implementation of the classic **Snake Water Gun** game using Python. In this game, the user plays against the computer, and the winner is determined based on predefined rules.

## Rules

* 🐍 Snake drinks Water → Snake wins
* 💧 Water douses Gun → Water wins
* 🔫 Gun kills Snake → Gun wins
* Same choices → Draw

## Features

* User vs Computer gameplay
* Random computer choice generation
* Winner determination based on game rules
* Simple and beginner-friendly Python code

## Technologies Used

* Python 3
* Random module

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd snake-water-gun
   ```

3. Run the program:

   ```bash
   python main.py
   ```

## Example

```text
Enter your choice (s/w/g): s
Computer chose: w

You Win! 🏆
```
